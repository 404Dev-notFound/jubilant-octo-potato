Quick Start Guide

Get up and running with CODECOLLAB in under five minutes. This guide covers the basics of workspace setup, repository connection, and starting your first AI-assisted sprint.

rocket_launch

Fast Onboarding

Import your existing GitHub or GitLab repositories with a single click. Our zero-config setup handles dependencies automatically.

smart_toy

AI Assistance

Enable autonomous agents to monitor your PRs and suggest structural improvements based on your team's unique coding style.

terminal 1. Installing CLI Tools

The CODECOLLAB CLI is the fastest way to interact with your workspace from your local environment. Use the following command to install it globally via npm:

Terminal — Bash 
npm install -g @codecollab/cli
cc auth login
cc project init --template node-standard
