How to Start CodeCollab Locally

Follow these steps to run the frontend and backend locally:

### 1. Start the Backend Database
Make sure Docker Desktop is running, then open your terminal in the project folder:
cd "c:\Users\Dell\Downloads\overall\dev\obsidian\projects and it's code\codecollab_v2"
Run this command to start the PostgreSQL database:
docker-compose up -d

### 2. View the Database (Optional)
To view the seeded project data in your browser, run:
npx prisma studio

### 3. Start the API Server
In a new terminal window inside the project folder, run the Node.js API server:
node server.js

### 4. Start the Frontend Website
In another new terminal window inside the project folder, run a local python server:
python -m http.server 8080

### 5. View the App
Open your web browser and go to:
http://localhost:8080/
